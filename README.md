# Cedar Entity & Schema Visualizer

A simple web-based tool for visualizing Cedar schemas and entities.

Try online on: https://anuk909.github.io/Cedar-Visualizer/
Recommended for PC/desktop viewing only.

## Features

- Load and visualize Cedar schemas and entities
- Display entity hierarchies
- View schema definitions
- Interactive entity exploration

## Usage

Recommended for PC/desktop viewing only.

1. Open `index.html` in a web browser
2. Load your Cedar schema and entities files from the `inputs/` directory
3. Explore the visualization

## Structure

- `index.html` - Main HTML page
- `script.js` - JavaScript logic
- `styles.css` - Styling
- `inputs/` - Directory for input files for example

## Requirements

No build process or dependencies required. Works in any modern web browser.
